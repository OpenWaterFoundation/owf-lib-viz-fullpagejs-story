*This page is designed to be a basic example of a possible story layout configuration, while walking through some of the elements on this page to explain how it can be utilized by others.

FullPage.js

The scrolling features of this page are created using the FullPage.js JavaScript library.

Currently, this is limited to only using the basic scrollable features of this library. There are several extensions available, that provide features such as horizontal scrolling and parallax effect. However, the extensions for the fullpage.js library require purchase. For the most part, a scrollable page is sufficient enough to accomplish what is needed, and the extensions can be added later, if necessary.

The current configurations of this page can be seen below. To learn more about these configurations view the documentation


                $(document).ready(function() {
                  $('#fullpage').fullpage({
                    anchors:['top','fullpage','text-pages','viz1', 'images', 'map', 'copyright'],
                    sectionsColor:['#9DB66B','','','','','','#5A5A5A'],
                    navigation:true,
                    navigationPosition:'right',
                    navigationTooltips:['OWF Stories','FullPage.js','Text Pages','Adding a Visualization', 'Background Images', 'Leaflet Map', 'Copyright'],
                    recordHistory:false,
                    menu:"#navbarResponsive"
                  });
                });

Text Pages

This page is an example of adding only text to a section. Text pages can be utilized to explain elements in the story layout and to set up context for the proceeding sections.

This page also makes use of the bootstrap grid system. Since the only thing added to this page is text, it is contained in a div with class='col-lg-12' which tells bootstrap to fill the entire width with the contents of this div. You will notice this content doesn't actually fill the entire width of the page. This is because the whole section is nested within a div class='container' which contains all the elements on this page inside a predefined width.

---------------------------------------------------------- This is an example of nesting elements inside a div with class='container-fluid' which will fill the elements spanning the whole width of the page. ----------------------------------------------------

Here is an example of setting one div inside the container to the class='col-lg-6' which tells bootstrap to make this div half the width of the container that it is nested inside.

This makes it easy to set two elements side by side on the page, which you will see utilized throughout the rest of this site when adding visualizations.

In general there are two text styling options. This text is nested within paragraph tags with no extra styling added to it.

This text, similar to the text above, has added style to it using class='lead'.
If you'd like you can always add new elements to the css stylesheet for more text styling options.



Adding a Visualization

This page gives an example of including a visualization with the use of iframe.

All of the contents on this page are nested within div class='container-fluid' to fill the whole width of the page. There are two divs.

One div class='col-lg-8' contains a single iframe which links to the gapminder visualization. This iframe has frameborder='0' set to remove the border around it. Having no border creates a more visually appealing layout, however there are benefits to seeing where the iframe starts and ends. Due to the nature of iframes and fullpage.js, if you wish to scroll to another page, up or down, the cursor must be outside of the iframe or it will not register with fullpage.

The other div contains the text on this page and has class='col-lg-4' to fill the remaining grid. This div also has class='text-right-custom' which is a custom style in the stylesheet which removes padding on the left side of the div so the text fits nicely next to the visualization. If you wish you can move the text div before the div containing the visualization iframe and specify class='text-left-custom', for the opposite effect.

Here is an example of adding a background image to the page. For the most part, using a custom image may require different placement of the text, different text color, and even different text size. For this reason it will probably require custom style being added to pages with a background image. This text is being centered utilizing bootstrap class='text-center'.

The background image is added to the nesting div class='col-lg-12' with added
style="background-image:url('local/path/to-file')" to apply background image. This div also has style='height:100vh' so the image fits the entire height of the page.

Leaflet Map
